# 个人记账系统

一个面向中文用户的本地单用户记账工具。数据保存在本机 SQLite 数据库中，支持支付宝、微信和历史 Excel 账单导入，提供月度分析、流水管理、公费报销跟踪与 Excel 备份。

## 特点

- 本地运行，无账号、无云端同步，账目默认保存在 `data/account_book.db`。
- 支付宝、微信账单自动解析、去重和过滤；重复导入不会重复写入。
- 支持账户、分类、消费标签、备注、交易对方、支付方式和报销状态。
- 流水列表采用本地表格组件，选择、全选和编辑草稿不会导致页面刷新。
- 收入、支出、过手往来和公费垫付分别统计，避免混淆个人收支。

## 功能说明

### 仪表盘

- 使用“年份下拉框 + 12 个月按钮”选择统计周期；无流水的月份不可选择。
- 展示个人收入、个人支出、结余和交易笔数。
- 单独展示当月过手转出、过手转入、待报销总额和已收回报销金额。
- 提供年度分类支出汇总表、月度收支趋势、当月分类支出饼图和账户支出饼图。
- 提供公费垫付报销清单，便于查看待报销与已结清记录。

个人收支统计会自动排除“公费垫付”“垫付报销”“过手转出”和“过手转入”。

### 流水列表

- 选择年份和月份后，一次显示该月全部流水，不分页。
- 关键词搜索备注、分类和交易对方；输入关键词时可搜索整个数据库，清空后回到当前月份。
- 支持账户、收支、分类、最低金额和最高金额组合筛选。
- 支持勾选行、表头全选、批量删除和单条修改。
- 支持“修改流水”进入当前可见流水的整表编辑模式；只有点击保存才写入数据库。
- 编辑模式支持单击编辑、下拉选择、消费标签批量设置、复制粘贴、键盘导航和取消确认。
- 本地表格支持固定表头、横向滚动、列宽调整、列顺序拖动、复制单元格内容和日期显示切换。
- 调整后的列布局保存在当前浏览器；恢复默认布局需要二次确认。

当存在未保存的整表编辑时，页面导航、年份、月份、搜索和字段筛选会被锁定，需先保存或取消。

### 账户、分类、标签与报销

常用账户包括支付宝、微信、现金、银行卡、美团月付和手动录入，也可保留历史导入账户。

支出分类包括生活费用、伙食费用、交通出行、休闲娱乐、办公学习、外出旅游、医疗保健、服饰鞋帽、非日用品、其它支出、过手转出和公费垫付。收入分类包括工资收入、生活费收入、转账收入、银行利息、兼职收入、其它收入、过手转入和垫付报销。

- 支出可设置消费标签：生存刚需、品质生活、自我投资、人情往来。
- 收入标签始终为空。
- 选择“公费垫付”后，报销状态自动为“待报销”，可改为“已结清”。
- 待报销公费行显示淡红色，已结清公费行显示淡绿色。
- 收支切换后，不兼容分类会自动清空；取消公费垫付后，报销状态会自动清空。

### 导入账单与备份

#### 支付宝与微信

- 支持支付宝 CSV/XLSX、微信 CSV/XLSX 官方账单。
- 自动识别表头、编码和常见字段名称，统一映射到账户、收支、金额、分类、备注、对方和支付方式。
- 使用 `import_hash` 去重，重复导入自动跳过。
- 导入后展示新增、重复跳过、自动过滤和预览结果。

自动过滤规则：

- 支付宝：余额宝收益发放、花呗自动还款、银行卡定时转入。
- 微信：转入零钱通；同时检查备注和映射后的分类。

被自动过滤的记录不会写入数据库，但会保存在当前浏览器会话的历史过滤记录中，可在导入页查看或清除。刷新页面、关闭浏览器或重启服务后，这些会话记录会清空。

#### 历史 Excel

解析器支持导入旧版记账 Excel 的支出与收入流水表，包括 `支出表`、`收入表`、`Sheet1`、`Sheet2` 与 `Sheet3`。支持“备注 发红包的人”备注列，并保留支出标签；日期仅含日期时自动补为当天零点。

导入页可导出数据库中的全部流水为 Excel 备份。

### 手动记账

- 填写日期、账户、收支、金额、分类、标签、备注、对方和支付方式。
- 金额按正数输入，系统按收支类型自动保存为收入正值或支出负值。
- 手动录入也使用去重规则，避免重复提交。

## 数据规则

| 项目 | 规则 |
| --- | --- |
| 金额 | 数据库中收入为正、支出为负；界面输入始终为正数。 |
| 去重 | `import_hash` 是导入去重键；手工修改不会改写它。 |
| 公费垫付 | 只能是支出，报销状态为待报销或已结清。 |
| 垫付报销 | 只能是收入，用于登记实际收回款项。 |
| 数据库 | 所有账目保存在本地 `data/account_book.db`。 |

## 快速开始

### 环境要求

- Windows 10/11
- Python 3.10 或更高版本

首次使用请在项目目录安装依赖：

```powershell
python -m pip install -r requirements.txt
```

### 启动方式

推荐双击 [启动.bat](启动.bat)。脚本会关闭占用 8501 端口的旧服务，以后台方式启动应用，并自动打开浏览器。

也可使用命令行：

```powershell
python -m streamlit run app.py --server.port 8501 --server.headless true
```

或使用启动器自动分配端口：

```powershell
python launcher.py
```

默认地址为 `http://localhost:8501`。

## 打包为桌面程序

在 PowerShell 中执行：

```powershell
.\build_exe.ps1
```

构建结果位于 `dist\CodexAccountBook\`。打包脚本会包含本地表格组件资源。

## 项目结构

| 文件或目录 | 说明 |
| --- | --- |
| `app.py` | Streamlit 页面、会话状态、筛选、表单、图表和本地表格交互。 |
| `db.py` | SQLite 建表、迁移、CRUD 与统计聚合。 |
| `parser.py` | 支付宝、微信和历史 Excel 账单解析、去重与自动过滤。 |
| `local_transaction_editor.py` | 本地表格 Streamlit 组件入口。 |
| `components/transaction_editor/` | 本地表格前端资源。 |
| `launcher.py` | 自动端口、浏览器打开与冻结模式启动器。 |
| `启动.bat` | Windows 一键后台启动脚本。 |
| `build_exe.ps1` | PyInstaller 打包脚本。 |
| `data/account_book.db` | 本地 SQLite 账本数据库。 |
| `更新记录.md` | 已交付功能的按日期更新记录。 |

## 注意事项

- 导入前建议先导出 Excel 备份。
- 自动过滤记录仅为浏览器会话级复核信息，不是永久数据。
- 删除流水和恢复默认列布局属于不可逆操作，操作前请确认。
- 同一台电脑上不要同时启动多个占用同一端口的应用实例。
- 如需了解开发规范和架构约束，请阅读 `AGENTS.md`。
